<h1 align="center">UserSort</h1>

<p align="center">
Search, filter and sort user data dynamically
</p>

<p align="center">
<a href="https://usersort-proto.vercel.app/">Live Demo</a>
</p>

<hr>

<h3>Features</h3>

<ul>
  <li>Case-insensitive name search</li>
  <li>Filter users by role</li>
  <li>Sort users by age (ascending and descending)</li>
  <li>Debounced search for performance</li>
  <li>Highlight matching search text</li>
  <li>Responsive user interface</li>
</ul>

<hr>

<h3>Tech Stack</h3>

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>Vanilla JavaScript</li>
</ul>
